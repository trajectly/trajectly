# Security Policy

## Supported Versions

Security fixes are provided for the latest published release on PyPI and the current `main` branch.

## Reporting a Vulnerability

Please report vulnerabilities privately through GitHub Security Advisories:

<https://github.com/trajectly/trajectly/security/advisories/new>

When possible, include:

1. A clear description of the issue.
2. Impact and affected components.
3. Reproduction steps or a minimal proof of concept.
4. Suggested mitigation, if known.

Please do not open public issues for security vulnerabilities.
